# pixieweb: Access PX-Web Statistical Data from R

A pipe-friendly R client for PX-Web statistical APIs. Provides a
search-then-fetch workflow for discovering and downloading data from
national statistics agencies (SCB, SSB, Statistics Finland, etc.) using
a consistent tibble-based interface.
